# 参考文献ページ 引き継ぎ

## 担当ファイル

- `pages/references.html`

## 役割

NVセンター関連文献を、水中の泡/樹形図風UIで整理するページです。左側のツリーからカテゴリを選び、右側に代表図風カードと文献カードを表示します。

## 見るべきコード位置

- 文献データ: `references`
- レビュー論文: `reviews`
- カテゴリ定義: `categories`
- 左ツリー座標: `treeNodes`
- 代表図風カード: `shotSets`, `shotLabels`, `renderScreenshotGrid`
- 文献カード描画: `renderPapers`
- カテゴリ選択: `selectCategory`

## 現在の分類

- `NVセンター`
- `レビュー論文`
- `ダイヤモンド`
  - 窒素
  - 空孔
  - 生成方法
- `形成`
  - 空孔生成
  - 窒素注入
  - 低エネルギー照射
- `応用`
  - ナノダイヤモンド
  - バルクダイヤモンド

## 重要仕様

- 論文の実スクリーンショットを入れる場合は、著作権的に丸ごと再配布しないよう注意し、必要なら自作の代表図風サムネイルに置き換える。
- ユーザーは「トップデータと思う分かりやすい図」を並べたい意図です。実画像を入れる場合も、論文名、目的、手法、解決課題、未解決課題、日本語アブスト要約を添える設計にしてください。
- 水中の泡のような背景と、左半分の枝分かれツリー感を維持します。

## 改修候補

- 各論文に DOI / URL / PDFメモ / 図サムネイル / 日本語要約を追加する。
- `references` をHTML内配列から `assets/data/references.json` に分離する。
- フィルタ、検索、重要度S/A/B表示を強化する。

## 確認方法

- 左ツリーの各ノードを押して、右側の件数と文献リストが変わる。
- スマホ幅でツリーが横スクロールできる。
- 文献カードが画面外にはみ出さない。
